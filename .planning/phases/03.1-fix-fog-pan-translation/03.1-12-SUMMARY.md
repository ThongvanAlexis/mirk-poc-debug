### Rule 1 (auto-fix bug) — `rootBundle.loadString` fails on .frag binary blob

- **Found during:** Tasks 1+2 (running `fog_shader_uniforms_test.dart` for the new source-grep sub-test).
- **Issue:** Initial draft of the FOG-18 source-grep sub-test used `await rootBundle.loadString('assets/shaders/atmospheric_fog.frag')`. Runtime error: `FormatException: Unexpected extension byte (at offset 40)`. Root cause: Flutter compiles `.frag` files into binary IPLR shader blobs at build time; the asset bundle returns the binary blob, not the source text.
- **Fix:** Replaced `rootBundle.loadString(...)` with `File('assets/shaders/atmospheric_fog.frag').readAsStringSync()` (mirrors the existing pattern in `fog_tile_period_invariant_test.dart` which works on the source-text file in the project tree). Removed `setUpAll(TestWidgetsFlutterBinding.ensureInitialized)` since dart:io File reads don't need the binding initialised.
- **Files modified:** `test/infrastructure/mirk/shader/fog_shader_uniforms_test.dart`.
- **Verification:** Sub-test runs against the source-text file and asserts active-code formulation correctly.
- **Committed in:** `24d0917` (Tasks 1+2 atomic commit).

### Rule 1 (auto-fix bug) — initial worldMeters bound ceiling too low

- **Found during:** Tasks 1+2 (running `fog_meter_decomposition_test.dart` for the worldMeters bound assertion).
- **Issue:** Initial draft set the worldMeters precision-safe ceiling at 25_000 m. The test failed at z=10 lat=0° (equator) where mpp ≈ 152.88 m/raw_px → worst-case worldMeters ≈ 300_703 m. Plan's "≤ ~6200 m at z=15" wording assumed Melun lat (48.5°) where mpp is smaller due to cos.

Documented derivation in the constant docstrings.
- **Files modified:** `test/presentation/widgets/fog_meter_decomposition_test.dart`.
